Analysis description

The ChocAn software/terminal should be computerized to enable the provider and operator to directly interact with it.
For the provider, the terminal will collect the data for the service reports by prompting the provider for necessary
information (i.e. service number, provider id, member id, and etc). The terminal will prompt the operator for necessary
information. With the collected data, the operator can update member and provider records. All data will be collected
at the ChocAn Data Center where each report will be processed and be sent out (i.e. when manager requests or end of the
week progress report). To interact with the terminal or data itself, the terminal users should use a computer, keyboard,
and mouse. The provider should also have a card reader for the members to "input" their data. When the users interact
with the terminal, they will prompt the terminal to contact the ChocAn Data Center in order to receive neccesary data.
