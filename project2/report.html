Analysis description

The ChocoAn software/terminal should be computerized for the provider and operator to be able to directly interact with it.
For the provider, the terminal will walk through the collection of service report through prompting the provider with
necessary information (i.e. service number, provider id, member id, and etc). The same goes for the operator except the
operator will be dealing with the management of each member and providers and updating their status when receiving data
from the system at the ChocoAn Data Center. The overall data will be all collected at the ChocoAn Data Center where each
report will be processed and be sent out (i.e. when manager requests or end of the week progress report). To interact with
the terminal or the data overall, the user should use a computer, keyboard, and mouse. The provider should also have a card
reader for the members to "input" their data. When the users interact, they will prompt the terminal to contact the ChocoAn
Data Center in order to receive neccesary data.
